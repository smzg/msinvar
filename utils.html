Return \((q)_n=\prod_{i=1}^n(1-q^i)\).

msinvar.utils.poch(a, q, n)[source]¶

Return the Pochhammer symbol \((a;q)_n=\prod_{i=0}^{n-1}(1-aq^i)\).

If n is a list, apply poch() to all entries and take the product.
